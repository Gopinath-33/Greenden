# 🌿 Greenden - Plant Store

A clean, modern landing page for an online plant store, showcasing indoor and outdoor plants with a smooth shopping-inspired layout.

🔗 **Live Demo:** [https://gopinath-33.github.io/Greenden/](https://gopinath-33.github.io/Greenden/)

## 📋 Overview

Greenden is a front-end e-commerce style website for a plant business. It highlights featured products, store benefits, and customer reviews to give visitors a complete browsing experience.

## ✨ Features

- 🏡 **Hero Section** – Eye-catching banner with "Shop Now" and "Watch Video" call-to-actions
- 📊 **Trust Stats** – Displays 160+ plant species and 100+ happy clients
- 🛍️ **Best Sellers** – Product showcase with images, names, and prices (Purple Blossom, Peace Lily, Lavender Bloom, Pink Bouquet)
- 💎 **About Section** – Highlights store strengths: Large Assortment, Fast Delivery, 24/7 Support
- ⭐ **Customer Reviews** – Testimonial section for social proof
- 📩 **Newsletter Signup** – Subscription form to capture leads
- 📱 **Responsive Design** – Optimized for mobile, tablet, and desktop

## 🛠️ Built With

- **HTML5** – Page structure
- **Tailwind CSS** – Utility-first styling and responsive layout
- **JavaScript** – Interactivity

2. Navigate to the project folder
   ```bash
   cd Greenden
   ```
3. Open `index.html` in your browser
